### **Quick setup** — if you’ve done this kind of thing before

[ Set up in Desktop](x-github-client://openRepo/https://github.com/Lcsbs/tech-summary)

**or**HTTP/SSSH

```
git@github.com:Lcsbs/tech-summary.git
```



Get started by [creating a new file](https://github.com/Lcsbs/tech-summary/new/main) or [uploading an existing file](https://github.com/Lcsbs/tech-summary/upload). We recommend every repository include a [README](https://github.com/Lcsbs/tech-summary/new/main?readme=1), [LICENSE](https://github.com/Lcsbs/tech-summary/new/main?filename=LICENSE.md), and [.gitignore](https://github.com/Lcsbs/tech-summary/new/main?filename=.gitignore).

### …or create a new repository on the command line

```
echo "# tech-summary" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:Lcsbs/tech-summary.git
git push -u origin main
```

### …or push an existing repository from the command line

```
git remote add origin git@github.com:Lcsbs/tech-summary.git
git branch -M main
git push -u origin main
```
